# IQ-React 2 Startup Pack

Webpack 4 + Babel 7 + React 16 新版前端初始包  
使用新版 Webpack 體驗極速的 bundle 速度!

## Install

```
cd iq-react2
npm install
```

## Dev Mode

```
npm run start
```

## Prod Mode

```
npm run build
```

## Reference

本專案升級自 [IQ-react](http://192.168.1.136/SideProject/IQ-react)，如果你已經在使用它，請繼續使用，不需要進行更新
